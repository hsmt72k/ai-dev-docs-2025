## OpenAI Embeddings の設定

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>

#### 1. OpenAI アカウントにログイン

OpenAI アカウントにログインします。

OpenAI developer platform<br />
https://platform.openai.com/docs/overview

</Step>

<Step>

#### 2. 新しい API キーを作成

「Create new secret key」をクリックして、新しい API キーを生成します。名前（例: "Test Key"）を設定し、必要に応じて制限を追加します。

</Step>

<Step>

#### 3. API キーをコピー

作成された API キーはこのタイミングでしか確認できないため、必ずコピーして安全な場所に保存してください。

</Step>
</Steps>

### 2. Astra DB との統合

<Steps>
<Step>

#### 1. Astra DB の統合セクションへ移動

Astra DB のポータルにログインし、左側の「Integrations（統合）」セクションを選択します。

</Step>

<Step>

#### 2. OpenAI 統合を追加

「Add Integration」をクリックし、以下を入力します。

- **APIキー名**: API キーを識別するための名前（例: "Test Key"）
- **APIキーの値**: OpenAI から取得した API キー
- **データベースの選択**: 統合するデータベース（例: "Test DB YouTube"）を選択します。最大10個までデータベースを追加可能です

</Step>

<Step>

#### 3. 統合の完了

「Add Integration」をクリックすると、統合が有効になり、選択したデータベースと API キーがリンクされます。

</Step>
</Steps>

### 3. ベクトルデータベースの準備

<Steps>
<Step>

#### 1. 新しいデータベースを作成

Astra DB ポータルで「Create Database」をクリックし、サーバーレス型またはベクトル型データベースを選択します。
名前やプロバイダー（例: Google Cloud）を設定してデータベースを作成します。

</Step>

<Step>

#### 2. 埋め込みデータの挿入

映画プロットなどのテキストデータをベクトル化してデータベースに保存します。
Astra DB では埋め込みモデル（例: OpenAI）を使用して自動的にベクトル化が行われます。

</Step>
</Steps>

### 4. 統合後の操作

統合が完了すると、新しいデータベースが API キーとリンクされます。
これにより、埋め込みデータの保存や検索が可能になります。
必要に応じて既存のデータベースに統合を追加することもできます。

### まとめ

OpenAI Embeddings と Astra DB の統合は非常に簡単です。
API キーの取得から統合設定まで数ステップで完了し、生成 AI アプリケーションで必要なベクトル化されたデータ管理が可能になります。
この仕組みは映画プロットなどの検索や推薦システム構築にも役立ちます。
