これから、基礎的な AI RAG アプリをステップ・バイ・ステップで作成してく。

まずは、RAG とは何か？から始まり、RAG を実現するフレームワークとして LangChain について理解する。

そして、開発環境をセットアップしてから、基本的なチャット画面の構築にトライしてみる。

チャットの機能自体はゼロからスクラッチで作るのではなく、OpenAI の API を利用する。

画面ができたら、いわゆるバックエンド、サーバ側で API エンドポイントを作成していく。

作成する API を、順を追って１つずつ機能を拡張していき、複数の API ルート作成に挑戦するエクササイズを行なう。

最初のエクササイズはシンプルな回答をする Chat API の作成。

次のエクササイズは、履歴を考慮した Chat with History API の作成。

さらに、キャラクタを持ったユニークな Chat with Personality API の作成にも挑む。

エクササイズの最後は、ファイルをデータベースに見立てて回答を行なう、Chat with JSON API を作成する。

そしてエクササイズが終わった後には...、何かが待っている。
