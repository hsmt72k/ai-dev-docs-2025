`ai`、`@ai-sdk/openai` と `openai`、`@opentelemetry/api` パッケージを追加する。

```bash title="Terminal"
bun add ai @ai-sdk/openai openai @opentelemetry/api
```

`ai` パッケージである、Vercel AI SDKは、
Vercel 社が開発しているオープンソースソフトウェア（OSS）である。

Vercel 社はこのハンズオンのプロジェクトでも利用している Next.js フレームワークも OSS で開発している。

Vercel は、AI SDK を Apache 2.0 ライセンスで公開しており、
開発者が AI 駆動の Web サービスやユーザインタフェース（UI）を構築するために提供している。

詳しくは Vercel の公式ドキュメントをご覧ください。

AI SDK | The AI Toolkit for TypeScript<br />
https://sdk.vercel.ai/docs/introduction

git add .
