このハンズオンでは、基礎的な AI RAG アプリをステップ・バイ・ステップで作成します。
**RAG** とは、**R**`etrieval-`**A**`ugmented` **G**`eneration` の略で、情報検索と生成モデルを組み合わせた技術です。
このアプリケーションを通して、さまざまな技術やツールを学びながら、実際に動作するアプリを作成していきます。

![](/02-hands-on/01-basic-rag-app/00_overview_of_app/00_overview_of_app_banner.png)

## アプリの開発の流れ

### 1. **RAG の基本概念と LangChain の理解**

- まずは、RAG が何であるかを理解します
- 次に、RAG を実現するためのフレームワーク **LangChain** を学びます

### 2. **開発環境のセットアップ**

- 開発環境を整え、アプリケーション開発に必要なツールをインストールします

### 3. **基本的なチャット画面の構築**

- ゼロからスクラッチで作成するのではなく、OpenAIのAPIを使用して、基本的なチャット画面を作成します

### 4. **バックエンドの作成**

- サーバ側で API エンドポイントを作成し、実際に動作するチャット機能を構築します

### 5. **API の拡張**

- 初めはシンプルな **Chat API** を作成しますが、その後、機能を拡張していきます
- 複数の API ルートを作成するエクササイズも行います

### 6. **次のステップ**

- **Chat with History API**: ユーザーのチャット履歴を考慮したAPI
- **Chat with Personality API**: 特定のキャラクターを持たせたチャットAPI
- **Chat with JSON API**: JSONファイルをデータベースとして利用し、回答するAPI

### 7. **エクササイズの後に**

- 最後のエクササイズを終えた後には、次のステップに進むための準備が整います
